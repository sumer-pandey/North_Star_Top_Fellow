# Prototype Testing — North Star

## Methodology
- 5 participants (Tier-1 + Tier-2 early professionals)
- Survey via Google Form
- Tasks: rate clarity, suggest improvements, confirm usage intent

## Suggestions
- Show ROI% & salary uplift → added  
- Limit to 3–5 courses → added  
- Accreditation badges → added  
- Streak ring & badges → added  
- Budget filters → added  
- Orientation stepper → added  

## Results
- Clarity rating ↑ 3.4 → 4.6 (post changes)  
- 100% willing to use after iteration  

📌 *Reference: Google Form user test data (internal).*
