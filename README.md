## 🎯 Problem
Millions of Indian professionals want to upskill but face:
- Course overload + choice paralysis
- No clear ROI on learning investments
- Low consistency (completion <10%)  
*(Source: World Economic Forum, Future of Jobs 2025 — [Report PDF](https://reports.weforum.org/docs/WEF_Future_of_Jobs_Report_2025.pdf))*

## 💡 Solution
North Star ensures **decision → consistency → ROI → proof**:
- ROI-first shortlist (3–5 courses only)
- 7-day starter plan → quick activation
- Focus mode + streak badges → habit loop
- Certificates with ROI tag + project proof

## 📊 Key Outcomes
- Completion ↑ <10% → 35–40%
- Free→Paid conversion ↑ 2–3x
- Recruiter visibility ↑ 2–3x  
*(Sources: NASSCOM, Kantar 2023, internal testing)*

## 🛠️ System Design
- **Frontend:** PWA (React/Next.js), Tailwind
- **Backend:** FastAPI/Node microservices (User, Catalog, ROI, Roadmap, Learning, Certificate, Payments)
- **Data:** Postgres, Redis, Neo4j, OpenSearch, Redshift/Snowflake
- **Infra:** AWS (CloudFront, MediaConvert, Cognito, S3, Lambda)
- **Observability:** Jaeger, Prometheus, Grafana, Sentry
